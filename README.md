# vue-facebook-beta
A project emulating some features of facebook, live socket chat, post CRUD, profile page. Utilizing Node.js, mongodb and vue.js

after building, navigate to /client and enter 

npm run dev

navigate to /server

enter 

nodemon server.js

visit 127.0.0.1:8080 in browser
